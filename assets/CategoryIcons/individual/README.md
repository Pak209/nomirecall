# Nomi category icons

Each category has two transparent 270 x 270 PNG files:

- `category-<name>-light.png` for light mode
- `category-<name>-dark.png` for dark mode

Categories: Tech, Fitness, Trading, Music, Ideas, Coding, Projects, Travel,
General, Business, Agents, and Personal.

In an Xcode image set, assign the `light` file to Any Appearance and the
`dark` file to Dark Appearance. Render them as Original Image rather than
Template Image so the supplied purple/lavender strokes are preserved.
